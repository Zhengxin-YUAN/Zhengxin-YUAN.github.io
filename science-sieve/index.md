# 土工筛分法相关参考规范

<!--more-->
{{< admonition abstract>}}
土体的颗粒大小和颗粒级配在土木工程中，特别是在岩土工程中，扮演着重要的地位。<font color="blue">**筛分法**</font>是最传统也是最普遍使用的用来测定粒径分布的方法。本文总结了一些相关的实验规范供大家参考。
{{< /admonition>}}

## 3. 总结

经过仔细地分析与考虑，基于以下两点原因：

1. 保证数据点间距相同

2. 尽可能多地获得数据点

选择方孔筛，且粒径分别为：
2 mm; 1.25 mm; 0.8 mm; 0.6 mm; 0.45 mm; 0.3 mm; 0.2 mm; 0.15 mm; 0.105 mm; 0.076 mm.
